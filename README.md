# yippee.cafe

Public website for [Yippee Cafe](https://yippee.cafe): playful apps, games, music tools, and tiny internet experiments cooked fresh.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

This is a static Next.js site. Production output is exported to `docs/` for GitHub Pages deployment, with `CNAME` copied into the export.
